## Color Palette Reference

### Blue (Info) - Price from List
- **Border**: `#2196F3` (solid 3px)
- **Background**: `rgba(33, 150, 243, 0.05)` (5% opacity)
- **Icon**: MudBlazor `Color.Info`
- **Text**: MudBlazor `Color.Info`
- **Usage**: Indicates automatic price resolution from price list

### Orange (Warning) - Manual Override
- **Border**: `#FF9800` (solid 3px)
- **Background**: `rgba(255, 152, 0, 0.05)` (5% opacity)
- **Icon**: MudBlazor `Color.Warning`
- **Text**: MudBlazor `Color.Warning`
- **Chip**: MudBlazor `Color.Warning` with Text variant
- **Usage**: Alerts user that price has been manually changed

### Gray (Default) - Default Price
- **Border**: None
- **Background**: Default
- **Icon**: MudBlazor `Color.Default`
- **Text**: MudBlazor default text color
- **Usage**: Neutral state when using product default price

## Accessibility Features

1. **Tooltips**
   - 300ms delay to avoid interfering with navigation
   - Clear, descriptive text
   - Keyboard accessible

2. **Color Contrast**
   - All text meets WCAG AA standards
   - Icons provide semantic meaning beyond color
   - Border + background + text = triple reinforcement

3. **Helper Text**
   - Always visible (not hidden)
   - Doesn't rely on hover state
   - Screen reader friendly

4. **Icons**
   - Semantic meaning (PriceCheck, Edit, Info)
   - Paired with text labels
   - Consistent size (Small)
