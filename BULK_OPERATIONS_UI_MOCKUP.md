## Common UI Patterns

### Color Scheme (MudBlazor Default)
- **Primary**: Blue (#594AE2) - Buttons, selections
- **Success**: Green (#00C853) - Success indicators, checkmarks
- **Warning**: Orange (#FF9800) - Partial success alerts
- **Error**: Red (#F44336) - Error messages, failed counts
- **Default**: Gray (#757575) - Secondary text, borders

### Typography
- **Dialog Title**: Typo.h6, Bold
- **Section Headers**: Typo.subtitle2, Medium
- **Body Text**: Typo.body1, Regular
- **Field Labels**: Typo.body2, Medium
- **Captions**: Typo.caption, Regular
- **Results Numbers**: Typo.h5, Bold

### Spacing (MudBlazor)
- Dialog padding: 24px
- Section margin bottom: 16px
- Field spacing: 8px vertical
- Button spacing: 8px horizontal
- Alert margin: 16px bottom

### Components Used
- **MudDialog**: MaxWidth.Large (implicit)
- **MudForm**: Built-in validation
- **MudGrid**: 12-column responsive
- **MudTextField**: Variant.Outlined
- **MudNumericField**: Variant.Outlined, Format="N2"
- **MudSelect**: Variant.Outlined
- **MudRadioGroup**: Color.Primary
- **MudButton**: Color.Primary (confirm), default (cancel)
- **MudProgressLinear**: Indeterminate, Color.Primary
- **MudAlert**: Severity-based colors
- **MudPaper**: Elevation="2" for cards
- **MudExpansionPanels**: For error details
