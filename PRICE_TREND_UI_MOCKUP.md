## Component Breakdown

### 1. Existing Fields Section
- **Prezzo Predefinito**: Numeric input for default price
- **Aliquota IVA**: Dropdown for VAT rate selection
- **IVA Inclusa**: Checkbox for VAT inclusion

### 2. Price Trend Chart Section (NEW)

#### Statistics Cards
**Left Card - Purchase Prices (Blue)**
- Title: "Prezzo Acquisto Medio"
- Large display: Average purchase price (e.g., โฌ10.50)
- Subtitle: Price range (e.g., Range: โฌ10.00 - โฌ11.00)

**Right Card - Sale Prices (Green)**
- Title: "Prezzo Vendita Medio"
- Large display: Average sale price (e.g., โฌ15.75)
- Subtitle: Price range (e.g., Range: โฌ15.00 - โฌ16.50)

#### Margin Card (Full Width)
- Title: "Margine Medio"
- Display: Margin in euros and percentage (e.g., โฌ5.25 (33.3%))
- Color: Green for positive margin, Red for negative

#### Line Chart
- **X-axis**: Dates in dd/MM format
- **Y-axis**: Price values in euros
- **Blue Line**: Purchase prices over time
- **Green Line**: Sale prices over time
- **Legend**: Shows which line represents which price type
- **Height**: 350px
- **Interactions**: Hover to see exact values

#### Footer
- Shows the year of data being displayed (e.g., "Anno 2025")

## Responsive Design

### Desktop (md and up)
- Statistics cards in 2 columns (50% each)
- Full-width chart
- Margin card full-width

### Mobile (xs-sm)
- Statistics cards stacked vertically (100% width)
- Chart adjusts to container width
- Maintains readability on small screens

## Color Scheme

- **Primary**: Blue (Purchase prices)
- **Success**: Green (Sale prices, positive margins)
- **Error**: Red (negative margins)
- **Info**: Light blue (Purchase price statistics)
- **Secondary**: Gray (borders and text)

## Interaction States

- **Hover**: Chart lines highlight on hover
- **Data Points**: Show tooltip with exact price and date
- **Loading**: Spinner animation
- **Empty**: Informational alert message

## Integration with Product Detail

The component appears as part of the tab system:

```
[Informazioni Generali] [Prezzi e Finanza] [Classificazione] [...]
                              โ
                      Currently selected
```

When user clicks "Prezzi e Finanza" tab:
1. Existing price fields load (already implemented)
2. Price Trend Chart loads asynchronously
3. Shows loading spinner while fetching data
4. Displays chart once data arrives
5. Data is cached for the session
